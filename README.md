# Resplenda

Resplenda is a full-stack cosmetics management web application designed to help users organize products, track routines, receive reminders, and receive personalized recommendations based on their preferences.

The project focuses on clean system design, secure authentication, and a polished, dashboard-style user experience.

---

## Features

- Secure user authentication (login and registration)
- Personalized skin quiz to capture user preferences
- Product browser with category filtering and search
- User inventory with open dates and expiry tracking
- Custom routines with reorderable products
- Automated reminders for product expiry and maintenance tasks
- Responsive, modern UI designed for daily use
